Website Tiệm Bán Kem F4

Giới thiệu
- Đây là đồ án giữa kỳ của môn Web 2 trong chương trình đào tạo đại học. Dự án xây dựng một website bán kem với giao diện người dùng thân thiện và hệ thống quản trị cơ bản.

Công nghệ sử dụng
- HTML
- CSS
- JavaScript
- PHP

Tính năng chính
- Trang người dùng: xem danh sách sản phẩm, chi tiết sản phẩm, giỏ hàng, thanh toán.
- Trang quản trị (admin): quản lý sản phẩm, đơn hàng, người dùng.
- Giao diện responsive, dễ sử dụng trên cả desktop và mobile.

Nhóm thực hiện
